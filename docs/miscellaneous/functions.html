isMongoDbIdValid
isMongoDbIdValid(id)

Checks if a given string is a valid MongoDB ObjectId.

This function uses Mongoose's ObjectId.isValid method to determine if the provided string is a valid representation of a MongoDB ObjectId. MongoDB ObjectIds are 24-character hexadecimal strings that are unique identifiers for documents in MongoDB collections.

Parameters :
Name Optional Description
id No
  • The string to check for validity as a MongoDB ObjectId.
Returns : boolean
  • Returns true if the string is a valid MongoDB ObjectId, false otherwise.
